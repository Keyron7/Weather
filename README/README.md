# 汤姆天气（TomWeather）
TomWeather是一款基于Android的天气预报软件，基于《安卓第一行代码 第二版》的酷我天气（CoolWeather），并对其进行了相应的更改，它具备查看全国的省市县、查询城市天气、自由切换城市、手动更新天气等功能，背景图片由必应提供，API接口由和风天气提供。
